# Trainer Portrait Images

Drop trainer portrait images here. They'll appear in the small circles on Ash's Journey progress screen.

## File Names (case-sensitive)

**Gym Leaders:**
- `brock.png` - Brock (Pewter City)
- `misty.png` - Misty (Cerulean City)
- `surge.png` - Lt. Surge (Vermilion City)
- `erika.png` - Erika (Celadon City)
- `koga.png` - Koga (Fuchsia City)
- `sabrina.png` - Sabrina (Saffron City)
- `blaine.png` - Blaine (Cinnabar Island)
- `giovanni.png` - Giovanni (Viridian City)

**Elite Four:**
- `lorelei.png` - Lorelei (Ice Master)
- `bruno.png` - Bruno (Fighting Master)
- `agatha.png` - Agatha (Ghost Master)
- `lance.png` - Lance (Dragon Master)

**Champion:**
- `gary.png` - Blue/Gary (Champion)

## Image Requirements

- Format: PNG recommended (JPG also works)
- Aspect ratio: Square (1:1) recommended
- Will be displayed at 80×80px in circles
- Transparent backgrounds look best
